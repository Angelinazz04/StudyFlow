# StudyFlow
StudyFlow – A modern student productivity platform featuring task management, note-taking, calendar planning, Pomodoro focus sessions, analytics, GPA tracking, and social collaboration.

## ✨ Features

### 🔐 Authentication System
- User registration
- User login/logout
- Local storage based authentication
- Session persistence

### 📊 Dashboard
- Academic overview
- Weekly planner
- Study statistics
- Productivity insights
- Quick access widgets

### 📚 Course Management
- Create, edit, and delete courses
- Color-coded course organization
- Track course-related tasks and notes

### ✅ Task Manager
- Create assignments and tasks
- Priority levels
- Due dates
- Completion tracking
- Filtering and sorting options

### 📝 Notes Workspace
- Create and manage notes
- Course-linked notes
- Search and filtering
- Rich note organization

### 📅 Calendar System
- Monthly view
- Weekly view
- Event creation
- Assignment scheduling
- Academic planning

### ⏱️ Pomodoro Focus Timer
- Focus sessions
- Short breaks
- Long breaks
- Study session tracking
- Course association

### 📈 Analytics
- Study hour statistics
- Task completion analytics
- Productivity tracking
- Study streak monitoring
- Performance charts

### 🎯 Premium Features
- GPA Calculator
- Academic achievements
- Goal tracking
- Advanced statistics

### 👤 Profile System
- Student profile
- Academic overview
- Personal achievements
- Study progress tracking

### ⚙️ Settings
- Profile customization
- Avatar selection
- Theme switching
- Goal configuration

### 👥 Social Features
- Student directory
- Friend requests
- Messaging system
- Shared notes
- Academic networking
